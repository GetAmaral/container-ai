## 1. PERFIL TECNICO DO SITE

- Framework: React 18 + Vite + TypeScript
- UI: Tailwind CSS + shadcn/ui (componentes)
- Roteamento: React Router v7 (SPA - Single Page Application)
- Backend: Supabase (Auth, PostgreSQL, Edge Functions)
- Pagamento: Hotmart (checkout externo, abre em nova aba)
- Deploy: Docker + Nginx
- CDN: Cloudflare

IMPORTANTE: Por ser SPA (React Router), a navegacao entre paginas NAO recarrega
o site. Isso significa que o pixel base so dispara PageView UMA VEZ no load
inicial. Precisamos de um listener especial para SPA.

## 4. GARGALOS E FALHAS TECNICAS

### GARGALO 1: SPA sem PageView em navegacao (CRITICO)
  - O pixel base so dispara PageView no primeiro load
  - React Router navega sem recarregar a pagina
  - SOLUCAO: Listener no React Router que dispara PageView a cada navegacao

### GARGALO 2: Checkout externo perde rastreio (CRITICO)
  - Hotmart abre em nova aba (window.open)
  - Pixel do site NAO rastreia o que acontece na Hotmart
  - Nao tem como saber se usuario completou a compra pelo pixel client-side
  - SOLUCAO: Conversions API (server-side) no webhook da Hotmart

### GARGALO 3: Sem banner de consentimento (CRITICO para LGPD)
  - Site tem pagina /cookies mas NAO tem banner ativo
  - Politica de cookies nao menciona cookies de marketing/tracking
  - SOLUCAO: Banner de consentimento + categoria "Marketing"

### GARGALO 4: Botao WhatsApp sem tracking (MEDIO)
  - window.open() redireciona antes de qualquer tracking
  - SOLUCAO: Disparar evento Contact ANTES do window.open()

### GARGALO 5: Scroll tracking inexistente (BAIXO)
  - Nao ha como saber se usuario viu a secao de pricing
  - SOLUCAO: Intersection Observer para secao #pricing
