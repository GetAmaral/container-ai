## DIAGNOSTICO

### Por que banco proprio alem do Facebook?
  - Facebook Events Manager mostra dados DELE, nao seus
  - Dados do Facebook tem delay de ate 72h
  - Voce nao controla a retencao (Facebook pode mudar regras)
  - Com dados proprios, voce pode:
    → Criar dashboards customizados
    → Cruzar dados de pixel com dados de pagamento
    → Auditar consentimentos LGPD
    → Exportar dados quando quiser
    → Manter historico independente

### Tabelas Criadas
  pixel_events        → Cada evento disparado (PageView, Lead, etc.)
  pixel_sessions      → Sessoes anonimas de visitantes
  pixel_consent_log   → Registro de consentimentos (LGPD obrigatorio)
  pixel_funnel_daily  → Metricas diarias agregadas

### Integracao com banco existente
  - As tabelas tem prefixo "pixel_" para nao conflitar
  - Usam referencia para auth.users quando usuario esta logado
  - RLS habilitado (seguranca)
  - Service role para Edge Functions, anon para tracking client-side
